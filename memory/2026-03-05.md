1. **Smart Rebrand** — sed replacement on `.ts/.js/.md/.css/.html` only, skip `package.json` deps and lockfiles. First attempt broke `pnpm install` because it renamed internal npm dependency names (`openclaw` → `mouse-platform` in `extensions/memory-core`). ### Key Lesson
**Never sed-replace identifiers in code files.** Even a "smart" string-only parser breaks on:
- Type annotations: `cfg: OpenClawConfig` → `cfg: Mouse PlatformConfig` (spaces in types)
- Class names in `.js`: `class OpenClawA2UIHost` → `class Mouse PlatformA2UIHost`
- Import paths: `../infra/tmp-openclaw-dir.js` → `../infra/tmp-mouse-platform-dir.js` (file doesn't exist)

**Correct approach:** Rebrand docs/UI only. Use runtime config for display name. Can pull upstream OpenClaw updates cleanly.
